This repository has the dataset underlying the findings described in the following paper:

Alsaleh, N., & Farooq, B. (2022). Is Shared On-Demand Public Transit Always Sustainable? A Focus on Efficiency, Environmental Footprint, and Social Equity and Inclusion.


This repository provides the data needed to reproduce the figures presented in the paper. It is worth noting that the detailed operational data for the existing crowdsourced ODT in Innisfil (Uber Transit) is third-party data that we cannot upload directly.
