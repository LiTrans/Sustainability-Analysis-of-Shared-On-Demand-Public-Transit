This repository has the dataset underlying the findings described in the following paper:

Alsaleh, N., & Farooq, B. (2023). Sustainability Analysis of On-Demand Public Transit Systems.


This repository provides the data needed to reproduce the figures presented in the paper. It is worth noting that the detailed operational data for the existing crowdsourced ODT in Innisfil (Uber Transit) is third-party data that we cannot upload directly. To access this dataset, researchers can contact Matthew Di Taranto, Senior Account Executive-Transit Partnerships, at (416-628-3234) or at (matthew.ditaranto@uber.com).
